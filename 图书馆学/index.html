图书馆学

图书馆学基础

P138

1.试述数据、信息、知识、文献之间的关系

P138

  1. 数据是对某种情况的记录，包括数值数据与非数值数据，它是能够被记录和储存在计算机等媒体上的已知事实
  2. 经过加工处理后形成的具有参考价值的数据就转化为信息，信息是具有普遍性的事物运动的状态和方式，是由事物发出的数据、信号中所包含的意义。信息是知识的原料
  3. 知识来源于信息，是被人类系统化后的信息的一部分。知识可以促进信息的再生，新的信息又可以转换为新的知识
  4. 文献是储存与传递知识的载体，它是知识的一种物化形态，是知识范畴中的显性知识部分

2.简述图书馆学研究对象与研究客体的关系

P139

图书馆学研究对象是图书馆学所要阐释一种社会的本质现象，而研究客体则是与这种本质现象发生密切联系的一个现象系统。 图书馆学的研究客体是客观知识、知识集合、知识受众以及相互之间的关系。就一门学科而言，其研究客体包含了研究对象，并以研究对象为其客体系统的核心

3.简述文献单元与知识单元有哪些区别

P141

文献单元：是专门记录和传递有知识的人工载体单元，如一本书、一篇文章等

知识单元：是知识系统中有实际意义的基本单位。如一个概念、公式等

前者具有“硬性”特点，后者具有“软性”特点；前者往往表现为知识体系，后者往往表现为知识点；前者是静止的，后者是衍变的；前者组织方法比较单一，后者组织方法则复杂多样。但文献单元与知识单元又是相互联系的，在许多方面又有着共同的特性

4.简述图书馆读者权力有哪些

P142

读者在获取知识活动中享有平等获取知识权，自由选择知识权，知识信息知情权，知识服务保障权，对知识服务机构的批评、建议和监督权等

P152

  1. 使图书馆工作由繁变简，由杂乱走向统一和规范

9.什么是信息资源建设？请说明信息资源建设的具体内涵

P153

信息资源建设是指人类对处于无序状态的各种媒介信息进行选择、采集、组织和开发等活动，使之形成可资利用的信息资源体系的全过程。信息资源建设的内涵包括三个方面：信息资源建设所针对的是处于无序状态的各种媒介的信息，信息资源建设活动的内容是对信息进行选择、采集、组织和开发，信息资源建设的目的是形成可资利用的信息资源体系。

10.什么是数字化信息资源？它包括了哪些类型的信息资源

P152

数字化信息资源是指数字化的形式，将文字、图像、声音、动画等多种形式的信息存储在光、磁盘等非纸质载体中，以光信号、电信号的形式传输，并通过计算机和其他外部设备再现出来的信息资源。它包括网络信息资源和单机信息资源。

信息资源组织

1.什么是信息组织？信息检索工具有哪些类型

P177

信息资源组织也称为信息组织，是根据使用的需要，以文本及各种类型的信息资源为对象，通过对其内容特征等进行分析、选择、处理、序化、并以适当的方式加以提供的活动。

对于检索工具的类型，可以有多种不同的区分方法。

按照检索工具的对象和特点，检索工具包括文献目录、索引、机读数据库和网络搜索引擎。

检索工具按照依据的标识特征，可以分为以信息资源的外部特征为检索依据，提供从资源形式出发进行检索的途径，和以表达信息资源主题内容的标识为依据，提供从内容角度进行检索的途径。

按照系统中所用的标识是否进行控制，检索工具可以分为受控检索系统与自然语言检索系统两种。

按照系统中标识组配的特点，检索系统可以分为先组式检索工具和后组式检索工具。

按照采用的形式，检索工具可分为卡片式、书本式、缩微式、电子方式等类型；按照信息资源对象，检索工具可分为图书、期刊、报纸、专利、标准、档案、图像、会议文献等类型。

3.什么是信息描述？其作用是什么

P179

信息资源描述，又称为文献著录，是指根据信息组织和检索的需求，对信息资源的主题内容、形式特征、物质形态进行分析、选择、记录和活动。

信息描述的作用是： 1. 用来确认和识别该资源对象 2. 提供信息资源位置的信息 3. 通过描述数据中提供检索点，方便用户对资源的检索和利用 4. 供用户对资源的价值进行判断，决定是否选用

5.信息资源分类及其作用是什么？分类法有哪些基本类型

P185

信息资源分类是指根据信息资源的内容属性和其他特征，将各种类型的资源分门别类地、系统地组织和揭示的方法。分类过程中，用来作为区分根据的属性或特征，称为分类标准。信息资源分类的作用包括：进行资源组织，建立分类检索工具，分类统计，兼容工具。

按照信息资源分类法的编制方式，通常可分为等级列举式、分面组配式、列举一组配式三种类型。

P186 P189

对分类法的结构组成，国内大致有两种划分方法：一种是按照各部分的功能，将分类法分为类目体系、标记符号、说明与注释、类目索引四个方面；另一种则是按照构成的形式，将其分为编制说明、主表、副表、类目索引等四个部分。

分类法类目之间的关系：

  1. 从属关系
  2. 并列关系
  3. 交替关系
  4. 相关关系

7.对分类标记有哪些基本要求？分类标记的种类、标记制度有哪些？什么是预留空号法、八分法、双位制、借号法

P187 P188

分类标记符合的要求：

  1. 简明性
  2. 表达性
  3. 容纳性
  4. 助记性

号码种类：

  1. 纯号码
  2. 混合号码

编号制度：

  1. 顺序标记制
  2. 层累标记制
  3. 顺序一层类标记制
  4. 分面标记制

标记技术：

  1. 八分法：亦称广九法，即在采用层累数字标记的情况下，当同位类超过10个，不足18个时，前9为以0~8表示，8后面的标记用两位数字表示一次划分，用于解决同为类的号码配置问题。
  2. 双位制：亦称集团表记法，即在采用层累数字标记的情况下，当同位类超过18个时，直接以两位数字表示一次划分，以解决号码的扩充问题。
  3. 借号法：在采用层累数字标记的情况下使用的一种灵活借用上、下位类号码的配号方法。通常在上一级号码较宽裕时，用上级号码标示下级类目，在下级类目超过10个但超过不多时，根据情况，借用9以外的下级号码进行扩充，使号码配置更加灵活。
  4. 预留空号法：指根据科学发展情况和类目设置的可能，在配号时，预先留下一些空号，供类目增补时使用。

9.主题法的特征是什么？主题法有哪些基本类型

P192

主题法是指直接以表达主题内容的词语作为检索标识、以字顺为主要检索途径的标引和检索信息资源的方法

特征：直接选用自然语言中的词语进行标引和检索；以字顺作为主要检索途径，字顺方式始终是它的主要排检依据；以特定的事物、问题、现象，即主题为中心集中信息资源；通过在主题词下设置用、代、属、分、参等多种参考系统等方式解释主题词之间的关系。

P194

类型：标题法、元词法、叙词法、关系词法、受控主题法与非控主题法

11.概要说明范畴索引和词族索引的作用和编制方法

P196

范畴索引是一种按照词汇所属学科或专业范畴编制的概略分类系统，是从分类角度查找叙词的辅助工具，范畴索引的作用：

  1. 便于从分类角度查找与某一范畴有关的叙词
  2. 可以作为概略分类的依据
  3. 是编表选词、处理词间关系的工具

词族，是指一组具有属分关系的叙词集合。词族索引是一种以词族为款目单元，按照款目词的字顺排列，可以从等级关系的角度查找叙词的索引，词族索引的作用：

  1. 可以从词族出发查词
  2. 可在计算机检索系统中，自动进行上位词登录
  3. 可通过等级关系限定词义

12.什么是轮排索引？轮排索引有哪些作用

P197

轮排索引，亦称轮排表，是将词表中的叙词按词素的字顺排列，是含有同一词素的叙词集中显示与一处的词汇表，是一种从词素的角度查找叙词的辅助工具。作用是：

  1. 可以从词素角度出发对叙词进行查找
  2. 利用按词素集中的叙词的特点，帮助使用者选择最合适使用的叙词
  3. 依据其字面成族的特点，进行词间关系处理

14.什么是后控词表？后控词表有何特点

P202

后控词表是一种在检索阶段进行控制的词表，这种词表不像传统的词控制表那样在标引阶段对标识实施控制，而是在检索阶段通过同义控制和相关词推荐方式提供检索帮助。

特点：

  1. 不承担标引功能，只用于检索控制，词汇控制不如先控词表严格
  2. 入口词丰富
  3. 动态性强
  4. 有较强的灵活性和自由度
  5. 具有面向文献和用户的特点
  6. 词间关系不同于传统词表

1.什么是书目？书目有哪些基本类型

P205

书目又称目录，他是著录一批相关文献，按照一定次序编排组织而成的一种揭示和报道文献信息的工具。

类型：

  1. 国家书目：是全面登记与报道一个国家近期或往昔出版物的文献总目
  2. 联合目录：是揭示和报导若干文献收藏单位的全部或部分藏书的目录
  3. 地方文献书目：是全面揭示和报道某一地区的自然、历史、社会文献的书目
  4. 个人著述书目：是记载作者的著述或兼收他人评价作者及其著述的文献书目
  5. 推荐书目：针对特定的读者，采用特定的标准选择图书推荐给读者的书目

2.国内外学者对目录学的研究对象有哪些不同的论点

P211 P212

  1. 图书说：认为目录学是研究图书的一门学科
  2. 目录说：主张目录学的研究对象是书目
  3. 图书和目录说：认为当代目录学的任务是辩章学术，考镜源流，推荐好书，指导阅读；研究对象首先是图书，其次是书刊目录
  4. 关系说：有两种说法：①认为目录学的研究对象是研究社会生活中记录图书与利用图书的关系②认为目录学是研究认识与揭示图书的规律的科学，认识图书和揭示图书的活动规律就是目录学研究的对象
  5. 矛盾说：认为目录学是研究书目工作形成和发展的一般规律的科学。

3.谈谈我国古代目录学发展的成就与特点

P215

先秦——两汉时期的目录学：

  1. 开创了官修书目的先例
  2. 为我国目录学的形成奠定了基础
  3. 创立了“七分法”的书目分类体系
  4. 创立了叙录体提要等揭示文献的方法

魏晋南北朝时期的目录学：

  1. 书目数量增加，类型增加，如佛经目录、私人书目出现
  2. 对书目的作用有了更充分的认识，订正闻见，炳然区分
  3. 对简明登记书目和提要书目两种体例进行了讨论
  4. 在书目方法方面，创立了“立传”、“明宗”、“总经序、记存亡、设附录、撰缘记”等多种方法
  5. 在书目分类方面，出现了六分发和四步法的竞争
  6. “流略”一词汇出现和“簙录”类的确立，说明目录学作为一门独立的知识系统已为人们所承认、重视

隋唐时期的目录学：

  1. 推荐书目的出现

两宋时期目录学：

  1. 专科目录的繁荣，索引的产生，目录著作的考证

元明时期的目录学：

  1. 创立了新的编排分类体系

清代目录学：

  1. 使用目录学成为“显学”

6.简述文摘的特点、类型和编纂方法

P222

文摘的特点：

  1. 替代性：文摘是对原文的一种替代
  2. 模拟性：文摘是原文的一种情报模型
  3. 转换性：包括语种的转换，表达方式的转换，等等。

类型：

  1. 从编撰目的和职能来看，大体分为普及性文摘和情报性文摘
  2. 根据文摘编者又可以分为作者文摘、学科权限文摘和专职文摘员文摘
  3. 根据文摘编写形式可分为文章式文摘和电报式文摘
  4. 根据文摘刊登的地方可分为同址文摘和非同址文摘
  5. 根据文摘的出版形式可分为单券式文摘和期刊式文摘

编纂方法：

  1. 确定选题和工作计划
  2. 文献的搜集与选择
  3. 摘要编写与款目著录

2.概述雕版印刷和活字印刷

P261

雕版印刷术是我国传统的“四大发明”之一。中国是世界上最早使用雕版印刷术的国家，时间可以推到唐朝初年，此后雕版印刷业逐渐发展，在宋元时期已经达到顶峰。

活字印刷术技术也是我国古人的发明。据记载，已知四种以上的印书活字，是中国人首先发明的。它们是：泥活字、木活字、铜活字、铅活字。

3.图书装帧的本质属性何在？需要遵循的基本原则是什么

P262

装帧之“帧”，一般指画幅，后用作专指画幅的量词单位。

图书装帧要遵循四项原则：

  1. 要体现思想性、艺术性和科学性的和谐统一
  2. 要具有可欣赏性的艺术特点
  3. 要坚持可行性与实用性的结合
  4. 要强调特色的连贯性和风格多样性

4.区别“阅”和“读”，并给予合理的解释和适当的发挥

阅的本义是逐一点看，意思是用眼睛一字不漏的仔细观看。

读的本义是照文字念、诵，通过用嘴出声念诵以达到对与字句意义的理解，如宣读、朗读、诵读。

广义的阅读，含义及其广泛，它包括了从不知到渐知、从泛知到追求深知的全过程。阅读是主体与客体相互作用的过程，是阅读主体实践活动与精神活动的一种体现。

侠义的阅读，涉及阅读客体，阅读主体和阅读本体，以及阅读学研究专题等内容。
